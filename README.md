# Hiro-Career

A personal career portfolio and interview preparation repository for Hiro Liu.

This repository is intended to keep career-related materials organized and reusable across future interviews.

## Structure

```text
Hiro-Career/
├── portfolio/              # HTML career portfolio
│   ├── index.html
│   ├── style.css
│   ├── script.js
│   └── assets/
├── speaker/                # English scripts and speaker notes
├── resume/                 # Resume files
├── projects/               # Detailed project notes
│   ├── DevicePort/
│   └── TrendMicro/
└── interview/              # Company-specific interview preparation
```

## Portfolio

Open the portfolio directly in a browser:

```bash
open portfolio/index.html
```

On Windows, double-click `portfolio/index.html` or open it with Chrome / Edge.

Keyboard navigation:

- Right / Down / Space: next slide
- Left / Up: previous slide
- Home: first slide
- End: last slide

## Suggested Git setup
